# Детекторы

## Частотный детектор
Частотный детектор использует комплексное умножение для определения мгновенной частоты входного сигнала. Детектор основан на методе дифференцирования фазы.

### Основной принцип работы:
- Комплексное умножение: Умножаем текущий отсчет сигнала на комплексно-сопряженный предыдущий:

```
product = signal[n] × conj(signal[n-1])
```
Аргумент этого произведения дает разность фаз между отсчетами.

- Вычисление частоты: Мгновенная частота вычисляется как:

```
f_inst = Δφ × Fs / (2π)
```
где Δφ - разность фаз, Fs - частота дискретизации.

### Ключевые особенности:
- Развертка фазы (Phase Unwrapping): Устраняет скачки фазы при переходе через ±π.
- Сглаживание (Smoothing): Экспоненциальное сглаживание для уменьшения шума.
- Альтернативная реализация PLL: Предлагает более устойчивое отслеживание частоты в условиях шума.
- Обработка блоков данных: Поддержка как поточного, так и блочного режимов.

### Параметры:
- sampleRate: Частота дискретизации входного сигнала
- alpha: Коэффициент сглаживания (0 = нет сглаживания, 1 = максимальное)
- Для PLL: bandwidth определяет скорость отслеживания

### Применение:
- Демодуляция ЧМ-сигналов
- Измерение частоты в системах связи
- Анализ спектра в реальном времени
- Слежение за несущей частотой

Детектор может работать с комплексными сигналами любой природы (I/Q данные SDR, аналитические сигналы и т.д.).
